Note: How To Add Or Remove Sections

To add a new section, copy one of the existing tab links at the top and one of the matching section blocks below, then paste both copies into the same pattern position.

When you do this, make sure the link target and the section name match exactly. For example, if the top link points to section-four, the section below it should also be named section-four.

To remove a section, delete both pieces for that section: the tab link at the top and the matching expandable section block below it.
