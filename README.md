The following directory contains the relevant code for the CidiLabs remediation project.

In the sample-cidilab-code/ directory, we have a list of individual HTML files containing sample code that is taken from real Canvas courses that use Cidilabs code elements. The files are named according to the courses from which they are taken, and the code snippets are real pieces of code used in those courses. 

The file directory name (for example, "homepage", or "accordian") denotes the Cidilabs component that is sampled in the corresponding HTML files within the directory.

In the sample-user-code directory, we have a list of HTML files that are our team's best representation at recreating the Cidilabs code using standard/free HTML elements. This is where the bulk of the work will be.

Our end goal is to create our own copy of every Cidilabs component that appears in each of our Canvas courses. For now, we are generally trying to analyze the files in these directories to understand the current context of our project.
